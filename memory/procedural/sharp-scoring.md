# SHARP 评分体系 — 2026-05-05 引入
# 基于 TheCalmCritic v2.0.0 BLIND-SCAN 框架

## 概述

SHARP 是五维量化评分体系，用于判断输出是否达到可交付标准。

## 五维结构

| 维度 | 全称 | 权重 | 最高分 |
|------|------|------|--------|
| B | Bias（偏见检测） | 5 | 5 |
| L | Logic（逻辑验证） | 5 | 5 |
| I | Information（信息审计） | 5 | 5 |
| N | Narrative（叙事解构） | 5 | 5 |
| D | Dependency（依赖分析） | 5 | 5 |
| **合计** | | **25** | **25** |

**通过阈值**: ≥ 18/25

## 各维度分项说明

### B-Bias 评分标准

| 分值 | 含义 |
|------|------|
| 5 | 未发现任何偏见痕迹，证据呈现平衡全面 |
| 4 | 存在轻微倾向性，不影响核心结论（P3） |
| 3 | 存在明显选择性呈现，需补充反面视角（P2） |
| 2 | 存在系统性偏见，结论可靠性显著下降（P1） |
| 1 | 存在严重偏见，结论基本不可信（P0） |
| 0 | 偏见问题严重到无法评估，或完全无法验证 |

### L-Logic 评分标准

| 分值 | 含义 |
|------|------|
| 5 | 推理链完整严谨，无逻辑谬误 |
| 4 | 推理链基本完整，存在轻微不严谨（P3） |
| 3 | 推理链存在断点或轻微谬误（P2） |
| 2 | 推理链存在显著逻辑谬误（P1） |
| 1 | 推理链存在根本性逻辑错误（P0） |
| 0 | 逻辑混乱到无法评估，或完全无法验证 |

### I-Information 评分标准

| 分值 | 含义 |
|------|------|
| 5 | 信息完整、来源多元、时效充足、不确定性已量化 |
| 4 | 信息基本完整，存在轻微时效或来源问题（P3） |
| 3 | 信息存在明显缺口或来源单一（P2） |
| 2 | 关键信息缺失或来源不可信（P1） |
| 1 | 核心数据完全缺失或严重过时（P0） |
| 0 | 信息状况无法评估，或完全无法验证 |

### N-Narrative 评分标准

| 分值 | 含义 |
|------|------|
| 5 | 纯逻辑论证，无叙事包装或情感干扰 |
| 4 | 存在轻微修辞装饰，不影响逻辑（P3） |
| 3 | 叙事包装部分掩盖逻辑弱点（P2） |
| 2 | 故事化或情感化显著削弱严谨性（P1） |
| 1 | 结论基本靠叙事或情感支撑，无实质逻辑（P0） |
| 0 | 叙事策略完全主导，无法评估逻辑；或完全无法验证 |

### D-Dependency 评分标准

| 分值 | 含义 |
|------|------|
| 5 | 所有依赖已声明、假设已检验、边界清晰 |
| 4 | 存在少量未声明假设，不影响核心结论（P3） |
| 3 | 存在显著未声明依赖或边界模糊（P2） |
| 2 | 核心结论依赖关键未声明条件（P1） |
| 1 | 结论建立在大量未验证假设之上（P0） |
| 0 | 依赖关系完全不明，无法评估；或完全无法验证 |

## 裁决等级

| 评分区间 | 等级 | 裁决 | 说明 |
|---------|------|------|------|
| 22-25 | A+ | PASS | 直接通过，品质卓越 |
| 18-21 | A | PASS | 通过，记录 P2/P3 供迭代参考 |
| 14-17 | B | CONDITIONAL_PASS | 有条件通过，需补充 P1 验证 |
| 10-13 | C | REJECT | 驳回，修正 P0/P1 后重新提交 |
| 0-9 | F | REJECT | 严重驳回，建议重新设计 |

## P0-P3 风险分级

| 级别 | 名称 | 描述 | 自动效果 |
|------|------|------|---------|
| P0 | 阻断级 | 根本性错误，关键信息缺失或严重偏见 | REJECT |
| P1 | 重大级 | 显著漏洞，需补充验证 | CONDITIONAL_PASS |
| P2 | 一般级 | 可改善瑕疵，降低可信度 | PASS_WITH_NOTE |
| P3 | 提示级 | 轻微优化建议 | PASS |
